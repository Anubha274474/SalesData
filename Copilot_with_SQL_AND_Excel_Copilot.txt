# Handling duplicate Values

How to Identify Duplicate Rows
A duplicate row means all values across columns (Customer_Name, Region, Product, Sales, Quantity, Order_Date) are identical. In your dataset, duplicates can occur because:

Multiple entries were logged for the same customer/product/date.

Some rows have missing values but still repeat the same pattern.

import pandas as pd

# Load dataset
df = pd.read_csv("Excel_dataset.csv")

# Remove duplicate rows
df_cleaned = df.drop_duplicates()

# Save cleaned dataset
df_cleaned.to_csv("Excel_dataset_cleaned.csv", index=False)

print("Duplicates removed. Cleaned dataset shape:", df_cleaned.shape)

Transformations Applied
Handle Missing Values

Customer_Name: Filled with "Unknown Customer" if blank.

Region: Filled with "Unknown Region" or mode (most frequent region) for that customer.

Product: Filled with "Unknown Product" if missing.

Sales: Converted "NA" → NaN. Imputed with median sales per product.

Quantity: Converted "two" → 2. "NA" or blanks replaced with median quantity (≈2–3).

Order_Date: Kept as is (valid dates).

Standardize Data Types

Sales → numeric (float).

Quantity → numeric (int).

Remove Duplicates

Exact duplicates across all columns dropped.

First occurrence kept.

# Question 10- Generate VBA code for cleaning data(removing duplicates, filling null values, format table).

Sub CleanSalesData()

    Dim ws As Worksheet
    Dim tbl As ListObject
    Dim lastRow As Long, lastCol As Long
    
    ' Set worksheet
    Set ws = ThisWorkbook.Sheets("Sheet1") ' Change to your sheet name
    
    ' Find last row and column
    lastRow = ws.Cells(ws.Rows.Count, 1).End(xlUp).Row
    lastCol = ws.Cells(1, ws.Columns.Count).End(xlToLeft).Column
    
    ' Remove duplicates (based on all columns)
    ws.Range(ws.Cells(1, 1), ws.Cells(lastRow, lastCol)).RemoveDuplicates Columns:=Array(1, 2, 3, 4, 5, 6), Header:=xlYes
    
    ' Fill null values
    With ws.Range(ws.Cells(2, 1), ws.Cells(lastRow, lastCol))
        .Replace What:="", Replacement:="Unknown", LookAt:=xlWhole   ' Fill blanks with "Unknown"
        .Replace What:="NA", Replacement:="Unknown", LookAt:=xlPart ' Replace "NA" with "Unknown"
        .Replace What:="two", Replacement:="2", LookAt:=xlWhole     ' Convert "two" to 2
    End With
    
    ' Format as table
    On Error Resume Next
    Set tbl = ws.ListObjects("SalesTable")
    On Error GoTo 0
    
    If tbl Is Nothing Then
        Set tbl = ws.ListObjects.Add(xlSrcRange, ws.Range(ws.Cells(1, 1), ws.Cells(lastRow, lastCol)), , xlYes)
        tbl.Name = "SalesTable"
    End If
    
    ' Apply table style
    tbl.TableStyle = "TableStyleMedium9"
    
    MsgBox "Data cleaning complete! Duplicates removed, nulls filled, and table formatted.", vbInformation

End Sub
What This Macro Does
Removes duplicates across all columns (Customer_Name, Region, Product, Sales, Quantity, Order_Date).

Fills missing/null values:

Empty cells → "Unknown"

"NA" → "Unknown"

"two" → 2

Formats the dataset as a table with a clean style (TableStyleMedium9).
